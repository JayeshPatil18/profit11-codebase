# Profit11 🚀
A fantasy stock market app designed to provide an engaging investment experience. Built using **Flutter & Firebase**, the app offers real-time stock contests, portfolio management, and secure transactions.

## Features 🏆
- 📈 **Live Stock Market Simulation** - Compete in stock market contests.
- 🎮 **Fantasy Contests** - Join leagues & earn rewards.
- 🔐 **Secure Transactions** - Integrated payment gateway.
- 📊 **AI-Powered Insights** - Personalized stock recommendations.
- 📅 **Real-time Updates** - Stock price fluctuations & news.

## Tech Stack ⚙️
- **Flutter** (UI/UX)
- **Firebase** (Auth, Firestore, Cloud Functions)
- **REST APIs & WebSockets** (Real-time stock data)
- **Payment Gateway Integration**

## My Role 👨‍💻
- Developed and optimized **Flutter UI** using **Riverpod** for state management.
- Implemented **real-time stock market updates** using WebSockets.
- Integrated **Firebase Authentication & Firestore** for seamless user experience.
- Built a **custom tutorial experience** using `tutorial_coach_mark` package.
